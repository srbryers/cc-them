# cc-them agents for FATHOMS art pipeline

Deferred agent profiles to author when the FATHOMS art-pipeline work needs them. The choice + ordering came out of a coverage-gap audit run on 2026-04-25 against the 33 existing roster while planning a multi-room tavern rebuild (Phase 16, S48+).

The principle: every agent must trace to a real practitioner with an auditable public record. Don't author one of these until you'd actually use it within the next sprint or two — they're work to research, and stale profiles mislead.

## Status

- **Tier 1** — authored 2026-04-25. See `profiles/christopher-alexander/`, `profiles/frank-ching/`, `profiles/marcus-rediker/`.
- **Tier 2** — deferred. Pick up when the relevant work begins.
- **Tier 3** — deferred to a later phase.

## Tier 2 — author when starting the work below

### `stuart-craig`
- **Domain:** Set / production design (film).
- **Trigger:** When the tavern interior moves past structural shell into furniture / dressing / "lived-in" detail. Probably begins with the first proper interior dressing sprint (post-Phase 16).
- **Why him:** Three-time Oscar winner (Harry Potter series, Dangerous Liaisons, The English Patient). Many published interviews about authenticity in fictional spaces — what makes a set "feel inhabited" vs "feel built."
- **Source notes:** Behind-the-scenes interviews on YouTube, *The Art of Harry Potter* book, AC Magazine + Production Design Magazine pieces. Sourcing should be doable in 1–2 hours.

### `vittorio-storaro`
- **Domain:** Cinematography, color-as-emotion, lighting-as-narrative.
- **Trigger:** When hero renders move from "scale check" to "publication shots." First trigger is when the tavern is structurally complete and we want a piece of art to lead Phase 16 close-out.
- **Why him:** Apocalypse Now, The Last Emperor. Documented philosophy of "writing with light," strong public corpus on color theory + the emotional language of lighting.
- **Source notes:** *Writing with Light* book series, AC Magazine archive, Storaro film festival lectures.

### `frank-thomas-and-ollie-johnston`
- **Domain:** Animation principles (the "12 principles of animation").
- **Trigger:** Phase that introduces character animation — Mixamo retargeting / hand-keyed expressive animation / cinematic shots. This will not be a near-term phase; flag for when characters cross from idle/walk into actual performance.
- **Why them:** Two of Disney's "Nine Old Men." *The Illusion of Life* (1981) is THE canonical reference work on animation; the 12 principles are cited in every animation curriculum.
- **Source notes:** *The Illusion of Life* book + extensive secondary sources. Joint profile because their work is co-authored throughout. Slug suggestion: `frank-thomas-ollie-johnston`.

## Tier 3 — author later, lower urgency

### `jane-jacobs`
- **Domain:** Urban / street / neighborhood scale design.
- **Trigger:** Nassau town buildout. Multiple buildings on a street, mixed program (residential + commercial + dock + tavern + market). Estimated several phases away.
- **Why her:** *The Death and Life of Great American Cities* (1961). "Eyes on the street," mixed-use, walking-distance fabric. Will keep us from building a sterile theme-park Nassau.
- **Source notes:** Book widely cited. Many interviews on YouTube. Solid sourcing.

### `inigo-quilez`
- **Domain:** Procedural rendering, signed distance functions, shader craft.
- **Trigger:** Procedural material variety (weathering, edge wear, noise that doesn't read as "noise") OR procedural prop variety (coral, palm leaves, rope coils with non-repeated geometry). Likely once the manual material library proves limiting.
- **Why him:** Pixar tools work, Shadertoy founder, prolific public posts on iquilezles.org with worked examples + math derivations.
- **Source notes:** iquilezles.org has full archive; Shadertoy + GitHub repos; many talks on procedural workflows.

## Notes on the audit

The roster covers strategy, engineering, game design, narrative, psychology, UX, growth, marketing, philosophy. **No coverage** of: physical-space design (architecture, interiors, set), visual production (cinematography, color theory, lighting), historical authenticity (period material culture), animation craft, or shader/procedural rendering.

Tier 1 picks up the most-immediate gaps for the building composition + period authenticity work. Tiers 2 and 3 wait for the pipeline to actually need them.

If a future audit reveals a gap not on this list — production designer for ships, costume designer for character work, or a music/sound-design agent for environmental audio — add it here with a trigger note before authoring.
